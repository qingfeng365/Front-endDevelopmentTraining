# 02 参数检查

## egg-validate

[https://github.com/eggjs/egg-validate](https://github.com/eggjs/egg-validate)


## 示例

```js
    const { ctx } = this;

    const paramRule = {
      id: 'string',
      code: 'string',
    };
    ctx.validate(paramRule);
```

检查失败,将自动触发错误


## postman

要设置 header 

"Content-Type" : "application/json"

否则会返回网页


## 返回错误结果

```js
{
    "code": "invalid_param",
    "message": "Validation Failed",
    "errors": [
        {
            "message": "required",
            "field": "id",
            "code": "missing_field"
        }
    ]
}
```

## 如果不需要触发错误

```ts
    const paramRule = {
      id: 'string',
      code: 'string',
    };
  const errors =  (this.app as any).validator.validate(paramRule, ctx.query);

```
