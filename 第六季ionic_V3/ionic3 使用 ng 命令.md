# ionic3 使用 ng 命令

```
npm install @angular/cli @types/jasmine --registry=https://registry.npm.taobao.org --disturl=https://npm.taobao.org/dist --sass-binary-site=http://npm.taobao.org/mirrors/node-sass
```

`/.angular-cli.json`

```
{
  "apps": [
    "root": "src"
  }],
  "defaults": {
    "styleExt": "scss"
  }
}
```
